API Documentation

Endpoints available for the Decentralized Identity Verification Tool.

  • POST /api/verify: Submits user documents for identity verification and returns verification status.
  • GET /api/status/{user_id}: Retrieves the verification status for a specific user.
  • GET /api/history/{user_id}: Returns the verification history for a specific user.
